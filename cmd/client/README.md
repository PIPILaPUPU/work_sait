# Скалодром - Клиентская часть

React + TypeScript + Vite приложение для сайта скалодрома.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

## Сборка для продакшена

```bash
npm run build
```

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
